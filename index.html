Intelligent Cost Optimization

Krolik Felix automatically routes tasks to the right AI model tier, saving you money without sacrificing quality.

Multi-Tier Routing

Automatically selects from Free, Cheap, Mid, or Premium tiers based on task complexity. Simple tasks use free models, complex ones get premium attention.

90% Cost Savings

Reduce AI API costs dramatically by using expensive models only when necessary. Free tier handles 60%+ of tasks, premium tier for the critical 10%.

Cascade Fallback

If a cheaper model fails, automatically escalates to the next tier. Ensures reliability while maintaining cost efficiency through smart retries.

History Learning

SQLite database tracks success/failure patterns. Routes similar tasks to models that performed well historically, improving accuracy over time.

TypeScript Native

Built for TypeScript developers with full type safety. Integrates seamlessly with tRPC, Prisma, Next.js and modern TypeScript stacks.

CLI & MCP

Use as CLI tool or Model Context Protocol server. Works with Claude Code, integrates into your development workflow seamlessly.

Frequently Asked Questions

Everything you need to know about Krolik Felix

Krolik Felix is an open-source TypeScript CLI tool that automatically routes AI tasks to optimal model tiers (Free, Cheap, Mid, Premium) based on complexity. It reduces AI API costs by 90% through intelligent multi-tier routing.

Felix analyzes each task's complexity and automatically selects the most cost-effective AI model:

  • Free tier (60% of tasks): formatting, refactoring, simple fixes
  • Cheap tier (25% of tasks): features, debugging, testing
  • Mid/Premium tiers (15% of tasks): architecture, complex algorithms

If a cheaper model fails, it automatically cascades to the next tier for reliability.

Krolik Felix is completely free and open source under the FSL-1.1 license (converts to Apache 2.0 on December 26, 2027). You only pay for the AI model APIs you consume, which are automatically optimized to reduce costs by 90%.

Felix supports all major AI providers:

  • OpenAI: GPT-4, GPT-4 Turbo, GPT-3.5 Turbo
  • Anthropic: Claude Opus, Sonnet, Haiku
  • Google: Gemini Pro, Gemini Flash
  • Open Source: Llama, DeepSeek, and more

Felix automatically manages API keys and routing across all providers.

No code changes required. Krolik Felix works as a CLI tool or MCP server:
