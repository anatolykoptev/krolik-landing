Common Questions

Everything you need to know about Krolik Felix

Krolik Felix is an open-source TypeScript CLI tool that automatically routes AI tasks to optimal model tiers (Free, Cheap, Mid, Premium) based on complexity. It reduces AI API costs by 90% through intelligent multi-tier routing.

Felix analyzes each task's complexity and automatically selects the most cost-effective AI model:

  • Free tier (60%): formatting, refactoring, simple fixes
  • Cheap tier (25%): features, debugging, testing
  • Mid/Premium tiers (15%): architecture, complex algorithms

Krolik Felix is completely free and open source. You only pay for the AI model APIs you consume, which are automatically optimized to reduce costs by 90%.

Felix supports all major AI providers:

  • Opus, GPT-4o, Gemini 1.5 Pro (Premium)
  • Haiku, GPT-3.5, Flash (Cheap)
  • Llama 3, DeepSeek (Local/Open Source)
